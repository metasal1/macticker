# Jup Bar

A floating macOS ticker bar for Jup.ag users. Track Solana tokens, click to trade, and stay in flow.

## Features
- Floating always-on-top ticker bar
- Hover to pause, click to open Jup
- Manage tickers (add/delete, pin, reorder)
- CSV import/export
- Price move alerts
- Custom RPC support

## Build
```bash
swift build
```

## Run
```bash
swift run
```

## DMG
The notarized installer is output to:
